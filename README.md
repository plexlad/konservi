# konservi
A simple, self-hostable archival and family history application built for
everyone.

## Description

Konservi is an open-source personal and collaborative history platform designed
with accessibility in mind. Most archival software is either too complex for
everyday users or locks your data behind proprietary formats. Archival solves
this by providing a clean, intuitive interface while ensuring you always own
your data.
The software is built for making it as straightforward to self-host and easy to
extend as possible.

## Core Features

- **User Permissions** - Role-based access control with viewer, editor, and
admin roles. Admins can invite collaborators and manage what each person can do
within a project.
- **Admin Panel** - Dedicated admin management with scoped permissions and audit
logging for all admin actions.
- **Web Interface** - A responsive, browser-compatible SPA that works across
common screen sizes and at least two major browser versions back.
- **WYSIWYG Editor** - A familiar rich-text editor supporting bold, italic,
headings, lists, and image embedding so writing entries feels natural.- **Export Formats** - Export your data as Markdown, GEDCOM, or XML so you are
never locked into one place.
- **Rigorous Backups** - Scheduled automatic backups with configurable
destinations, a visible backup status in the UI, and a full restore flow.
